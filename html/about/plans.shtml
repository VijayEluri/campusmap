Plans

Many parts of the application are not yet complete. This is a very brief outline of what we still plan to do. (This will all probably get done between the end of Summer Session I and the beginning of Fall 2005.)

  • A better base map. The current map is just the visitor map PDF. (It even still has the key in the upper right-hand corner!) We plan on using the colored visitor map.
  • Complete path data. The current path data is for demonstration only and is in no way complete. Once we obtain a final base map and create a scaled base image for use in the editor, we'll enter in the complete path data.
  • Google Maps-style scrolling. Who can resist?

...plus anything else that comes up.
