About UCSDMap

Moving the Map

  • The controls to the left of the map function very similarly to their conterparts in the plain CGI interface, and may be used to scroll and zoom the map as before.
  • The map may also be moved by clicking and dragging inside the main view.

Key Commands

Key Function
+ (plus) Zoom in
- (minus) Zoom out
Arrow keys Scroll the map in the given direction

Note: Not all key commands work in all browsers.

Browser Compatibility

Firefox, Internet Explorer, Safari, and Opera all work.

Unlike Google Maps, our interface does not require ActiveX on Internet Explorer.
